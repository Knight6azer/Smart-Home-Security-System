#### **For LinkedIn / Portfolio (Featured Project)**
**Title:** Smart Security System with Telegram Integration

**Description:**
I developed a low-cost, scalable security solution that bridges the gap between hardware sensors and modern communication platforms. This project uses an Arduino Uno to monitor physical spaces for motion (PIR) and proximity breaches (Ultrasonic).

**Key Innovation:**
Unlike traditional local alarms, I built a bridge between the embedded hardware and the cloud. I wrote a Python script that listens to the Arduino's serial output and instantly pushes critical alerts to a user's smartphone via Telegram. This ensures you know what's happening at home, even when you're miles away.

**Tech Stack:**
* **Hardware:** Arduino Uno, HC-SR04, PIR Motion Sensor, DHT11.
* **Software:** C++ (Embedded Logic), Python (Serial Communication & API requests), Telegram Bot API.

**Impact:**
* Successfully demonstrated real-time latency-free alerts.
* Implemented environmental monitoring (Temperature/Humidity) alongside security.
* Built a foundation for future Smart Home automation integration.
